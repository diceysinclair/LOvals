A romantic Valentine's Day website built with Java Spring Boot for your girlfriend!

## Features
- Beautiful animated landing page with floating hearts
- Countdown to Valentine's Day
- Personal love letter section
- Reasons why you love her
- Interactive surprise button
- Responsive design

## How to Run

### Prerequisites
- Java 17 or higher
- Maven

### Steps
1. Navigate to the VALs directory
2. Run: `mvn spring-boot:run`
3. Open browser to: http://localhost:8080

## Customization
Edit `HomeController.java` to change:
- `girlfriendName` - Her name
- Love letter content in `index.html`
- Reasons in the reasons section
- Colors and styles in `style.css`
